# Dokki Publish

Use the `dokki-publish` MCP server for published docs sites and custom domains.

Tools:

- `get_publish_site`
- `create_publish_site`
- `update_publish_site`
- `publish_resource`
- `unpublish_resource`
- `list_published_resources`
- `set_custom_domain`
- `remove_custom_domain`
- `check_domain_status`

Confirm before changing public visibility, domains, or unpublishing resources.
